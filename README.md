# Trending YouTube Videos Analysis

This project analyzes YouTube trending video data across multiple countries
using Python and pandas. The analysis was developed as part of the
Foundations of Computer Science course.

## Dataset
The dataset includes:
- CSV files containing trending YouTube videos for different countries
- JSON files containing video category metadata

## Tasks Performed
The project covers:
- Data loading and concatenation
- Data cleaning and preprocessing
- Tag analysis and aggregation
- Time-based analysis of trending dates
- Category metadata integration
- GitHub-based version control

## Tools Used
- Python
- pandas
- Jupyter Notebook
- Git & GitHub
